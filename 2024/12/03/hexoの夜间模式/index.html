取消夜间模式

  • 当某个标签不想使用暗黑/夜间模式时，可以添加 darkmode-ignore CSS 类到标签上。
  • 当然你也可以使用 isolation: isolate; CSS 样式来忽略暗黑/夜间模式。
  • 也可以使用这种 mix-blend-mode: difference CSS 样式来还原暗黑/夜间模式。
